# Ecommerce A/B Testing  — Анализ и проверка гипотез

### Описание проекта
В данном проекте проведён анализ результатов **A/B-теста e-commerce платформы**.  
Цель — определить, влияет ли новая версия сайта на **увеличение конверсии пользователей** по сравнению с текущей.


## Цели и задачи

**Цель:**  
Проверить статистически значимое изменение в конверсии между группами `control` и `treatment`.

**Задачи:**
1. Изучить структуру и качество данных (`ab_data.csv`, `countries.csv`)
2. Очистить и объединить датасеты
3. Посчитать ключевые продуктовые метрики (конверсия, размер выборок, доли стран)
4. Проверить гипотезу о различии конверсий между группами
5. Визуализировать результаты и сделать выводы

---

## Данные

Использовались два файла:

| Файл | Описание |
|------|-----------|
| `ab_data.csv` | Лог переходов пользователей: идентификатор, группа (`control`/`treatment`), флаг конверсии |
| `countries.csv` | Страна пользователя (для проверки возможного влияния региона) |

---

## Этапы анализа

1. **Загрузка и предварительный осмотр данных**
   - Проверка типов, пропусков, дубликатов  

2. **Очистка данных**
   - Удаление дубликатов по `user_id`
   - Исключение пользователей, которые встречались в обеих группах

3. **EDA (Exploratory Data Analysis)**
   - Распределение пользователей по странам  
   - Анализ различий в конверсии по странам и группам  
   - Визуализация: `barplot`, `countplot`, `piechart`

4. **A/B тестирование**
   - Формулировка гипотез:
     - **H₀:** Конверсии в группах одинаковы  
     - **H₁:** Конверсии различаются  
   - Применение Z-теста для пропорций 
   - Расчёт p-value и доверительных интервалов

5. **Интерпретация результатов**
   - Анализ статистической значимости  
   - Вывод о целесообразности внедрения новой версии


## Основные результаты

- Средняя конверсия группы **control** ≈ 12.0%  
- Средняя конверсия группы **treatment** ≈ 12.2%  
- **p-value > 0.05**, следовательно, различие **не является статистически значимым**  
- Внедрение новой версии **не улучшило метрику конверсии**
